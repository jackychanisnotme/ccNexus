# AINexus 第一阶段商业交付物

本目录用于承接 `docs/codex-goal-phase1-distribution.md` 中定义的四类付费验证产品和 30 天复盘。

## 文件

- [config-pack.md](config-pack.md)：配置包交付模板。
- [node-subscription.md](node-subscription.md)：节点订阅半自动交付模板。
- [deployment-service.md](deployment-service.md)：代部署服务范围、边界和验收。
- [api-entry.md](api-entry.md)：API 入口白名单交付、限制和风控。
- [30-day-validation-tracker.md](30-day-validation-tracker.md)：每日数据和 Go / No-Go 复盘。

## 第一阶段边界

- 不做 App Store 优先路线。
- 不做复杂 SaaS、多租户和公开 API marketplace。
- 不公开宣传 token 倒卖。
- 所有付款、交付和支持先走人工或半自动流程。
